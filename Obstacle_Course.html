Obstacle Course



About

One of the Robot projects the Computer Science Pathway students did was an Obstacle Course. There were groups of 2, usually with 1 senior and 1 junior, and each group came up with paths they want to take to drive their robot through. They then started to code and find the measurements for the degree turns and how low they want to go forward for they used a program called Python and this program is basically where they put all the code in.
